# Pre-v11 Archive

This archive records the Taro refactor state immediately before the `v11`
community/profile/tabbar expansion.

Archived baseline:

- branch: `codex/taro-app-refactor`
- app line: Taro `v10`
- scope: current tool page, BLE connection flow, local generation, export flow

Why this archive exists:

- preserve the pre-community single-page tool experience
- provide a rollback reference before adding tab navigation and server-backed community data

Reference files for the pre-v11 behavior are preserved in git history and this
archive marker documents the transition point.
